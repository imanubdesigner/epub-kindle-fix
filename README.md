# epub-kindle-fix

A browser-based tool to fix EPUB files for Amazon's **Send to Kindle** service.

👉 **[Use the tool](https://imanubdesigner.github.io/epub-kindle-fix/)** 👈

All 16 fixes run client-side in your browser. Nothing is uploaded to any server.

For the CLI version (Bash + Python), see the [main repository](https://github.com/imanubdesigner/epub-kindle-fix).
