👤 Original concept

The original idea, research, and core fixes come from innocenat's web-based tool kindle-epub-fix. This repository ports his work to the CLI and extends it with additional EpubCheck-style validations.

🧑‍💻 CLI port & extended fixes

CLI port, Python implementation, and additional fixes (9–16) by Manuel. This website and its client-side JavaScript EPUB fixer were also built by the same author.
